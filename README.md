# 🛍️ Saree Shop Website

This is a simple and responsive Saree Shop Website built using HTML and CSS.  
It showcases different types of sarees, collections, and estimated prices.

## 🚀 Features

- Navigation bar with sections (About, Contact, Collections, Models, Prices)
- List of available saree models
- Image gallery for collections
- Price table for different sarees
- About section with shop details
- Contact section with call, WhatsApp, and email options
- Hover effects and basic animations

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Sections Included

- 🏠 Home / Navigation
- 📋 Models List
- 🖼️ Collections (Images)
- 💰 Price Table
- ℹ️ About Us
- 📞 Contact Section

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser
